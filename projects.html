Credit Card Fraud Detection using Machine Learning

Advanced Diploma · AI Module (EIARI4A)

Developed and evaluated multiple machine learning models to detect fraudulent credit card transactions in real time. Working with a highly imbalanced Kaggle dataset of 284,807 transactions (only 0.17% fraud), the project compared six models — Logistic Regression, Random Forest, AdaBoost, CatBoost, XGBoost, and LightGBM — using stratified 70/15/15 train-validation-test splits and 5-fold cross-validation. XGBoost achieved the best result with a test AUC of 0.9778, addressing the core challenge of extreme class imbalance that causes naive models to miss all fraud cases.
